## [0.1.0-beta.1] - 2025-11-03

### Added
- Initial project structure
- Core embed API with `Splice.embed()` function
- Support for public and private harnesses via share tokens
- Customizable styling (CSS variables, JS options, web component attributes)
- Built-in theme presets (light, dark, high-contrast, blueprint)
- Interactive pan/zoom controls
- Optional BOM table display
- Controller API for programmatic control
- CDN distribution via jsDelivr and unpkg
- NPM package publication
